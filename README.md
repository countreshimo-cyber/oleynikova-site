# Сайт ЛОР-хирурга Олейниковой Юлии Александровны

Профессиональный многостраничный сайт для приема пациентов с интеграцией Telegram-бота для уведомлений о новых заявках.

## 📁 Структура сайта

Сайт состоит из следующих страниц:

1. **index.html** — главная страница с обзором всех услуг
2. **sinusit.html** — детальная страница о лечении хронических синуситов
3. **septoplastika.html** — страница об искривлении носовой перегородки (в разработке)
4. **vazomotornyj-rinit.html** — страница о вазомоторном рините (в разработке)
5. **hrap.html** — страница о лечении храпа и апноэ сна (в разработке)
6. **odontogennyj-gajmorit.html** — страница об одонтогенном гайморите (в разработке)
7. **zabolevaniya-uha.html** — страница о заболеваниях уха (в разработке)
8. **lor-procedury.html** — страница о ЛОР-процедурах (в разработке)

## 📱 Настройка Telegram-бота

### Шаг 1: Создание бота

1. Откройте Telegram и найдите [@BotFather](https://t.me/BotFather)
2. Отправьте команду `/newbot`
3. Введите название бота: `Олейникова ЛОР`
4. Введите username бота (например: `OleynikovaLorBot`)
5. Скопируйте полученный **токен** (выглядит как `123456789:ABCdefGHIjklMNOpqrsTUVwxyz`)

### Шаг 2: Получение Chat ID

**Вариант 1 (простой):**
1. Найдите бота [@userinfobot](https://t.me/userinfobot) в Telegram
2. Нажмите "Start"
3. Скопируйте ваш **Chat ID** (число, например: `987654321`)

**Вариант 2 (через ваш бот):**
1. Откройте созданного бота в Telegram
2. Нажмите "Start"
3. Откройте в браузере: `https://api.telegram.org/bot<ВАШ_ТОКЕН>/getUpdates`
4. Найдите `"chat":{"id":987654321` - это ваш Chat ID

### Шаг 3: Настройка в коде

В файле `oleynikova-lor.html` найдите строки (~870):

```javascript
const TELEGRAM_BOT_TOKEN = 'YOUR_BOT_TOKEN';
const TELEGRAM_CHAT_ID = 'YOUR_CHAT_ID';
```

Замените на:

```javascript
const TELEGRAM_BOT_TOKEN = '123456789:ABCdefGHIjklMNOpqrsTUVwxyz'; // Ваш токен от BotFather
const TELEGRAM_CHAT_ID = '987654321'; // Ваш Chat ID
```

### Шаг 4: Тестирование

1. Откройте `oleynikova-lor.html` в браузере
2. Заполните форму записи
3. Нажмите "Отправить заявку"
4. Проверьте Telegram - вам должно прийти уведомление

## 🌐 Размещение на GitHub Pages (бесплатный хостинг)

### 1. Создание репозитория

1. Зайдите на [GitHub](https://github.com)
2. Нажмите "New repository"
3. Название: `oleynikova-lor` (или любое другое)
4. Выберите "Public"
5. Нажмите "Create repository"

### 2. Загрузка файлов

**Через веб-интерфейс GitHub:**
1. Нажмите "uploading an existing file"
2. Перетащите файлы: `oleynikova-lor.html` и `doctor-photo.jpg`
3. Переименуйте `oleynikova-lor.html` в `index.html` (важно!)
4. Нажмите "Commit changes"

**Через Git (если установлен):**
```bash
git clone https://github.com/ВАШ_USERNAME/oleynikova-lor.git
cd oleynikova-lor
# Скопируйте файлы в эту папку
mv oleynikova-lor.html index.html
git add .
git commit -m "Initial commit"
git push
```

### 3. Активация GitHub Pages

1. Зайдите в Settings репозитория
2. Найдите раздел "Pages" в левом меню
3. В "Source" выберите "main" ветку
4. Нажмите "Save"
5. Через 1-2 минуты сайт будет доступен по адресу:
   `https://ВАШ_USERNAME.github.io/oleynikova-lor/`

## 🔧 Устранение неполадок

### Форма не отправляет сообщения

1. Проверьте правильность токена и Chat ID
2. Убедитесь, что вы нажали "Start" в боте
3. Откройте консоль браузера (F12) и проверьте ошибки
4. Проверьте, что бот не заблокирован

### Фото не отображается

1. Убедитесь, что файл фото находится в той же папке
2. Проверьте название файла (регистр важен!)
3. Проверьте путь в коде: `<img src="doctor-photo.jpg">`

### Сайт не открывается на GitHub Pages

1. Убедитесь, что файл называется `index.html`
2. Проверьте, что репозиторий публичный (Public)
3. Подождите 5-10 минут после активации Pages
