# ЧАСТЬ 2: РЕГИСТРАЦИЯ В GOOGLE

## ШАГ 2: Google Search Console

### 2.1. Регистрация

1. **Откройте:**
   ```
   https://search.google.com/search-console
   ```

2. **Войдите под Google-аккаунтом**

3. **Нажмите "Добавить ресурс"**

4. **Выберите "Домен"**, введите:
   ```
   oleynikovalor.ru
   ```

### 2.2. Подтверждение через DNS (REG.RU)

1. **Google покажет TXT-запись**, вида:
   ```
   google-site-verification=abc123xyz...
   ```

2. **Скопируйте её**

3. **Идём в REG.RU:**
   - Личный кабинет → Домены
   - Ваш домен → Ресурсные записи → Изменить
   - Добавить запись:
     - **Тип:** TXT
     - **Субдомен:** @
     - **Значение:** вставьте строку от Google
   - **Сохранить изменения!**

4. **Вернитесь в Google Search Console**

5. **Нажмите "Подтвердить"**

6. **Подождите 5 минут** — домен подтверждён! ✅

### 2.3. Отправка sitemap

1. **В Google Search Console слева нажмите "Sitemap"**

2. **Введите:**
   ```
   sitemap.xml
   ```

3. **Нажмите "Отправить"**

4. **Статус должен стать "Успешно"**

### 2.4. Запрос индексации

1. **Слева нажмите "Проверка URL"**

2. **Введите:**
   ```
   https://oleynikovalor.ru
   ```

3. **Нажмите Enter**

4. **Нажмите "Запросить индексирование"**

5. **Готово!** Google начнёт индексировать сайт в течение 1-3 дней

# ЧАСТЬ 3: РЕГИСТРАЦИЯ В ЯНДЕКСЕ

## ШАГ 4: Яндекс.Вебмастер

### 4.1. Регистрация

1. **Откройте:**
   ```
   https://webmaster.yandex.ru
   ```

2. **Войдите через Яндекс ID** (или зарегистрируйтесь)

3. **Нажмите "+ Добавить сайт"**

4. **Введите:**
   ```
   https://oleynikovalor.ru
   ```

5. **Нажмите "Добавить"**

### 4.2. Подтверждение

**Способ 1 — через meta-тег (проще):**

1. **Яндекс покажет meta-тег**, вида:
   ```html
   <meta name="yandex-verification" content="abc123xyz" />
   ```

2. **Скопируйте его**

3. **Добавьте в index.html:**
   - Откройте index.html в редакторе
   - Найдите секцию `<head>`
   - Вставьте meta-тег после других meta
   - Сохраните и загрузите на GitHub

4. **Вернитесь в Яндекс.Вебмастер**

5. **Нажмите "Проверить"**

6. **Готово!** ✅

**Способ 2 — через robots.txt:**

Яндекс может предложить добавить строку в robots.txt — это тоже работает.

### 4.3. Отправка sitemap

1. **В Яндекс.Вебмастере слева "Индексирование → Файлы Sitemap"**

2. **Нажмите "Добавить"**

3. **Введите:**
   ```
   https://oleynikovalor.ru/sitemap.xml
   ```

4. **Нажмите "Добавить"**

5. **Готово!**

### 4.4. Переобход страниц

1. **Слева "Индексирование → Переобход страниц"**

2. **Введите:**
   ```
   https://oleynikovalor.ru
   ```

3. **Нажмите "Добавить в очередь"**

4. **Яндекс проиндексирует сайт в течение 1-3 дней**

## ШАГ 9: Google Analytics (рекомендуется)

### Зачем:
- Узнаете, сколько человек заходит на сайт
- Откуда приходят (Google, Яндекс, соцсети)
- Какие страницы самые популярные

### Как подключить:

1. **Откройте:**
   ```
   https://analytics.google.com
   ```

2. **Войдите под Google-аккаунтом**

3. **Создайте ресурс** (property)

4. **Введите:**
   - Название: `Сайт Олейниковой`
   - URL: `https://oleynikovalor.ru`

5. **Google даст вам код**, вида:
   ```html
   <!-- Google Analytics -->
   <script async src="https://www.googletagmanager.com/gtag/js?id=G-XXXXXXXXXX"></script>
   <script>
     window.dataLayer = window.dataLayer || [];
     function gtag(){dataLayer.push(arguments);}
     gtag('js', new Date());
     gtag('config', 'G-XXXXXXXXXX');
   </script>
   ```

6. **Вставьте этот код:**
   - В index.html
   - В секцию `<head>`
   - Перед закрывающим `</head>`

7. **Загрузите на GitHub**

8. **Готово!** Через 24 часа начнут собираться данные
