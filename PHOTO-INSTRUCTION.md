# 📸 Как добавить фото на сайт

## Быстрая инструкция

### Шаг 1: Подготовьте фото

1. Сохраните ваше фото (файл `1000447096.png`)
2. **Переименуйте** его в: `doctor-photo.jpg`
3. Убедитесь, что файл называется именно так (строчными буквами, без пробелов)

### Шаг 2: Загрузите на GitHub

1. Откройте ваш репозиторий на GitHub
2. Нажмите кнопку **"Add file"** → **"Upload files"**
3. Перетащите файл `doctor-photo.jpg` в окно браузера
   (или нажмите "choose your files" и выберите его)
4. В поле комментария напишите: `Добавлено фото доктора`
5. Нажмите **"Commit changes"**

### Шаг 3: Проверьте результат

1. Подождите 2-3 минуты
2. Откройте ваш сайт (адрес из Settings → Pages)
3. Прокрутите до секции "Обо мне"
4. Фото должно появиться вместо плейсхолдера "Ю.А."

## ✅ Готово!

Код уже обновлён, фото встроено. Как только вы загрузите файл `doctor-photo.jpg` в репозиторий, оно автоматически отобразится на сайте.

## ❓ Частые вопросы

**Фото не появляется?**
- Проверьте название файла: должно быть `doctor-photo.jpg` (не `Doctor-Photo.JPG`)
- Файл должен быть в корне репозитория (не в папке)
- Подождите 3-5 минут и обновите страницу (Ctrl+F5)

**Фото слишком большое/маленькое?**
- Сайт автоматически подгоняет размер
- Оптимальное разрешение: 800-1200px по длинной стороне

**Хочу другое фото?**
- Просто загрузите новый файл с тем же названием `doctor-photo.jpg`
- GitHub перезапишет старый файл
