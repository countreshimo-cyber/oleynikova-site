## 📋 Пошаговая инструкция по установке

### Шаг 1: Создание репозитория на GitHub

1. Зайдите на https://github.com
2. Войдите в свой аккаунт (или зарегистрируйтесь)
3. Нажмите зелёную кнопку **"New"** (или знак "+" → "New repository")
4. Настройте репозиторий:
   - **Repository name:** `oleynikova-lor` (или любое другое имя)
   - **Description:** "Сайт ЛОР-хирурга Олейниковой Ю.А."
   - **Visibility:** выберите **Public** (обязательно!)
   - **НЕ ставьте галочки** на README, .gitignore, license
5. Нажмите **"Create repository"**

### Шаг 2: Загрузка файлов

**Способ 1 (простой) — через веб-интерфейс:**

1. На странице нового репозитория найдите "uploading an existing file" и нажмите
2. Перетащите **ВСЕ файлы из архива** в окно браузера:
   - 9 файлов .html
   - 3 файла .md
3. В поле "Commit changes" напишите: `Первая версия сайта`
4. Нажмите **"Commit changes"**

**Способ 2 (через Git) — для продвинутых:**

```bash
# 1. Клонируйте репозиторий
git clone https://github.com/ВАШ_USERNAME/oleynikova-lor.git

# 2. Перейдите в папку
cd oleynikova-lor

# 3. Скопируйте все файлы из архива в эту папку

# 4. Добавьте файлы
git add .

# 5. Сделайте коммит
git commit -m "Первая версия сайта"

# 6. Отправьте на GitHub
git push origin main
```

### Шаг 3: Активация GitHub Pages

1. На странице репозитория нажмите **"Settings"** (⚙️ справа вверху)
2. В левом меню найдите **"Pages"**
3. В разделе **"Source"** выберите:
   - **Branch:** `main`
   - **Folder:** `/ (root)`
4. Нажмите **"Save"**
5. Подождите 2-3 минуты

**Готово! Сайт опубликован!**

Ваш сайт будет доступен по адресу:
```

```

## 🔧 Если что-то не работает

### Сайт не открывается (404 ошибка)
- Проверьте, что файл называется `index.html` (не Index.html)
- Подождите 5-10 минут после активации Pages
- Проверьте, что репозиторий Public

### Стили не загружаются
- Все стили встроены в HTML, эта проблема не должна возникнуть
- Если возникла — очистите кэш браузера (Ctrl+Shift+R)

### Форма не отправляет в Telegram
- Проверьте, что вы написали боту первое сообщение
- Откройте консоль браузера (F12) и посмотрите ошибки
- Проверьте интернет-соединение

### Фото доктора не появляется
- Проверьте, что файл называется `doctor-photo.jpg`
- Файл должен быть в корне репозитория (не в папке)
- Подождите 2-3 минуты после загрузки
